# modinit

Initialize node packaged module.

## How to use

1. Create `.modinitrc` in your home directory.

```shell
$ modinit create
```

Ex:

```json
{
  "github": "morishitter",
  "author": "Masaaki Morishita",
  "mail": "ikasama3.0@gmail.com"
  "license": "MIT",
  "readme": "readme.markdown",
  "test": "tape",
  "travis": true
}
```
